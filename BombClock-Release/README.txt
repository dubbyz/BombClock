BOMB OVERLAY - Installation Instructions

1. Extract all files to your preferred location
2. (Optional) The digital timer font might not load unless its installed to your computer:
   - Double-click the Digital-7.ttf file
   - Click "Install" in the font preview window
3. Run BombClock.exe
4. Enter desired time.
5. Press "START CLOCK"
6. Enjoy!

Note: The application requires Windows and .NET 6.0 or later.
